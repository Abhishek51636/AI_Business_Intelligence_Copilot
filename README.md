# 📊 AI Business Intelligence Copilot

An AI-powered Business Intelligence and Data Analytics platform built using Python, Streamlit, Machine Learning, and OpenRouter LLM APIs.

The application enables users to upload datasets, clean data, generate visualizations, build machine learning models, generate reports, and interact with datasets using natural language through AI-powered analysis.

---

## 🚀 Project Overview

AI Business Intelligence Copilot is designed to simplify the data analytics workflow by combining:

* Data Cleaning
* Data Visualization
* Machine Learning
* Business Intelligence
* AI-Powered Insights
* Automated Reporting

into a single interactive dashboard.

---

## ✨ Key Features

### 📂 Dataset Management

* Upload CSV and Excel datasets
* Dataset preview
* Display dataset dimensions
* View column names and datatypes

### 🧹 Data Cleaning

* Missing value handling
* Replace categorical missing values with "None"
* Replace numerical missing values with 0
* Duplicate record removal
* Outlier detection using IQR method
* Download cleaned dataset

### 📈 Data Visualization

* Interactive charts
* Bar Charts
* Line Charts
* Scatter Plots
* Histograms
* Correlation Heatmaps

### 🤖 Machine Learning

* Linear Regression
* Random Forest Regressor
* Decision Tree Regressor
* Auto Model Recommendation
* R² Score Evaluation
* Mean Squared Error (MSE)
* Model Performance Analysis

### 🧠 Dataset Insights

* Dataset Profiling
* Missing Value Analysis
* Unique Value Analysis
* Statistical Summary
* Skewness Analysis
* Dataset Health Score
* Executive Summary Generation

### 🤖 AI Analyst

Powered by OpenRouter LLM API

Capabilities:

* Dataset Analysis
* Business Insight Generation
* Data Quality Assessment
* Machine Learning Recommendations
* Natural Language Dataset Chat
* AI-Powered Explanations

### 📄 Reporting

* PDF Report Generation
* Dataset Summary Export
* Cleaned Dataset Download

### 🎨 Modern User Interface

* Glassmorphism Design
* Gradient Backgrounds

* Responsive Dashboard Layout
